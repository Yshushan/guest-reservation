
## 访客登记时，前端传给后端的数据格式
### 员工 employee
```js
{
  name: String,
  phone: String
}
```
### 访客 guest
```js
{
  id: Number,  // 访客id，主键
  name: String, // 姓名
  address: String, // 单位
  gender: String, // 性别
  phone: String, // 手机号
  certificateType: String, // 证件类型
  certificateID: String, // 证件号码
  carID: String // 车牌号
}
```

### 携带物品 material
```js
{
  id: Number, // 物品id，主键
  count: Number, // 数量
  type: String, // 类型
  remark: String // 备注
}
```
### 登记表 register
```js
{
  id: Number, // 登记表id，主键
  employees: [employee], // 被访员工，可以是多个
  guests: [guest], // 到访人员，可以是多个
  material: [material], // 携带物品，可以是多个
  visitArea: String, // 到访区域
  visitTime: Date, // 到访时间
  visitType: String // 访问类型
}
```
## 查询到访记录时，后端传给前端的数据格式：
```js
{
  id: Number, // 每一条到访记录的主键
  status: String, // 申请状态
  submitTime: Date, // 提交时间
  visitType: String, // 访问类型
  employee: [ // 被访员工，可以是多个
    {
      name: String,
      phone: String,
    }
  ],
  guests: [ // 到访人员，可以是多个
    {
      id: Number,
      name: String,
      address: String,
      gender: String,
      phone: String,
      certificateType: String,
      certificateID: String,
      carID: String
    }
  ],
  materials: [ // 携带物品，可以是多个
    {
      id: Number,
      count: Number,
      type: String,
      remark: String
    }
  ]
}
```

## 到访区域
```js
{
  id: Number, // 区域id，主键
  name: String, // 区域名
}
```
