
## 访客登记时，前端传给后端的数据格式
### 访客 guest
```js
{
  // id: Number,  // 访客id，主键
  guestName: String, // 姓名
  guestAddress: String, // 单位
  guestGender: String, // 性别
  guestTelphone: String, // 手机号
  guestEmail: String, //
  certificateType: String, // 证件类型
  certificateNumber: String, // 证件号码
}
```

### 携带物品 guestBelongs
```js
{
  // id: Number, // 物品id，主键
  name: String,
  count: Number, // 数量
  type: String, // 类型
  remark: String // 备注
}
```
### 登记表 register
```js
{
  // id: Number, // 登记表id，
  userId: Number, // 被访员工id
  guests: [guest], // 到访人员，可以是多个
  guestBelongs: String, // 携带物品，可以是多个
  visitArea: String, // 到访区域
  visitDate: Date, // 到访时间
  visitType: String // 访问类型
  visitCarNumber: String // 车牌号
}
```
## 查询到访记录时，后端传给前端的数据格式：
```js
{
  id: Number, // 每一条到访记录的主键
  status: String, // 申请状态  待审核 ，通过， 拒绝
  createDate: Date, // 提交时间
  visitType: String, // 访问类型  // 
  carNumber: String
  employee:{
      userId： Number,
      userName: String,
      userPhone: String,
    },
  guests: [ // 到访人员，可以是多个
    {
      // id: Number,
      name: String,
      address: String,
      gender: String,
      telPhone: String,
      certificateType: String,
      certificateNumber: String,
      
    }
  ],
  guestBelongs: [ // 携带物品，可以是多个
    {
      // id: Number,
      name: String,
      count: Number,
      type: String,
      remark: String
    }
  ]
}
```
